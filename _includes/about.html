Front-end

I lean towards the backend but there's no denying how much bang for your buck a good front end gets you. I eventually got tired of my artistically talented family complaining about how my projects looked so I finally invested in putting in the time to make some slick user experiences.

Back-end

There's something so satisfying about being able to live on the backend. Being able to think about all the expected behaviour while writing tests, then jamming the code till they all pass. Not to mention being able to create beautifully designed code with tight abstractions that are flexible, modular and future minded.

Tools

One of my favourite things about working on personal projects is having the opportunity to be exposed to a range of tools at a more intimate level than I get at work. If there's something funky going on in the pipeline, I'm the one that's got to figure out what's going on.
